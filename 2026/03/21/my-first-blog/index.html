创建 GitHub Pages 仓库

GitHub Pages 会自动托管仓库里的静态文件，仓库命名有固定规则：

  1. 登录 GitHub，点击右上角「+」→「New repository」
  2. 仓库名称必须是：你的GitHub用户名.github.io（如 testuser.github.io）
  3. 勾选「Public」（必须公开），勾选「Add a README file」，点击「Create repository」
