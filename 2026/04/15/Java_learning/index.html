输入与输出

在 Java 中，输入和输出（I/O）可以通过不同的方式处理。最常见的方式是使用 System.out 和 BufferedReader。

System.out 输出

这是 Java 中最基本的输出方式：

System.out.println 用于打印一行文本并换行。

System.out.print 也可以用来输出，但不会换行。

BufferedReader 输入

当你需要从控制台读取输入时，可以使用 BufferedReader，它比 Scanner 更高效，尤其在处理大量数据时。

BufferedReader 通过 readLine() 方法读取一行输入。

Integer.parseInt 用于将字符串转换成整数。

基本数据类型与控制流

基本数据类型

Java 的基本数据类型包括：

  • int、float、double、char、boolean 等等。

与 C 相比，Java 的基本类型是严格的，不能随意转换类型。

条件判断与循环

Java 中的条件判断与循环语法与 C 十分相似：

  • if、else、for、while 控制流语法与 C 几乎一样。
  • Java 中没有 do-while 语句，但你可以使用 while 模拟。

数组在 Java 中是对象，长度在创建后固定，访问时通过 arr[index]，长度使用 arr.length（注意它是字段，不是方法）。

Java 是一个纯面向对象的编程语言。所有的代码都必须放在类中。

常用类库与 API

Java 提供了丰富的标准类库，以下是常用类别及其常用方法：

  • 字符串处理：String、StringBuilder、StringBuffer
  • 集合框架：ArrayList、HashMap、HashSet
  • 文件处理：File、BufferedReader、BufferedWriter
  • 日期与时间：LocalDate、LocalTime、LocalDateTime

字符串处理类

String

String 是 Java 中最常用的类之一，用于表示不可变字符串。

length()：返回字符串长度

charAt(int index)：返回指定位置字符

substring(int beginIndex, int endIndex)：获取子字符串

contains(CharSequence seq)：判断是否包含子串

equals(Object obj)：判断字符串是否相等

toUpperCase() / toLowerCase()：转大写/小写

StringBuilder 和 StringBuffer

可变字符串，常用于拼接。StringBuffer 线程安全，StringBuilder 更高效。

append(String str)：追加字符串

insert(int offset, String str)：插入字符串

delete(int start, int end)：删除指定范围字符

reverse()：反转字符串

集合框架类

List（ArrayList, LinkedList）

有序集合，允许重复元素。

ArrayList 基于动态数组，适合随机访问；LinkedList 基于链表，适合频繁插入和删除。

前提代码：

add(E e)：添加元素

get(int index)：获取元素

size()：元素数量

contains(Object o)：是否包含元素

remove(int index)：删除元素

Set（HashSet, TreeSet）

无序且不重复元素集合。

HashSet 基于哈希表，查询速度快；TreeSet 基于红黑树，元素会按排序规则保持有序。

前提代码：

add(E e)：添加元素

contains(Object o)：是否包含元素

remove(Object o)：删除元素

Map（HashMap, TreeMap）

键值对集合，键唯一。

HashMap 基于哈希表，适合快速查找；TreeMap 基于红黑树，键会按顺序排列。

前提代码：

put(K key, V value)：添加键值对

get(Object key)：获取值

containsKey(Object key)：是否包含键

remove(Object key)：删除键值对

文件处理类

File

用于表示文件和目录。

前提代码：

exists()：判断是否存在

delete()：删除文件/目录

BufferedReader, BufferedWriter

高效文件读写。

前提代码：

readLine()：读取一行

write(String str)：写入文本

日期与时间处理类

LocalDate, LocalTime, LocalDateTime

用于日期、时间、日期时间。

前提代码：

now()：获取当前日期/时间

plusDays(long days) / minusDays(long days)：加减天数

format(DateTimeFormatter formatter)：格式化

错误处理

在 Java 中，异常是通过 try-catch 块来处理的。

try 块用于执行可能抛出异常的代码，catch 块用于捕获并处理异常。
